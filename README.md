# AWS GuardDuty IR Lab

This project is a cloud security automation lab that detects GuardDuty findings and triggers an automated incident response workflow using EventBridge, Lambda, SNS, and EC2.

## Goal
Build an AWS incident response pipeline that can:
- receive GuardDuty findings
- trigger automated remediation
- alert the user
- isolate or stop affected EC2 instances
